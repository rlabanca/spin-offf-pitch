- Where a note conflicts with the financial plan or the founder's later confirmations, **the founder's confirmed answers win** (recorded in [`../../knowledge/`](../../knowledge/) and [`../../CLAUDE.md`](../../CLAUDE.md)).

  - Notes say corporate type S.A. → founder later confirmed **S.L. (Sociedad Limitada)**.
  - Notes mention €8/€10 exclusive-content subscription → financial model uses **€12/month**, which is canonical.
  - Notes don't anchor M1 to a calendar month → founder confirmed **M1 = January 2027**.
